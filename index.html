Compiling SASS

From root to specific file paths:

$ sass --watch path/to/sass/styles.sass:path/to/css/styles.css

From assets to css subfolder:

$ sass --watch sass/:css/

In the css folder:

$ sass --watch *

Variables

$var: value
  // Colors
  $green:     #819c31
  $navy:      #012a37
  $orange:    #ed8000
  // Links
  $link: $green
  $focus: #77bdd1
  $hover: #70c1da 
  $visited: #408295
  // Fonts
  $heading: 'CopseRegular',Archer,'Gill Sans',Georgia,sans-serif
  $copy: 'PTSansRegular','Trebuchet MS',Calibri,'Gill Sans',sans-serif
  $bold: 'PTSansBold','Trebuchet MS',Calibri,'Gill Sans',sans-serif
  $italic: 'PTSansItalic','Trebuchet MS',Calibri,'Gill Sans',sans-serif
